# PRD: AI 스터디 매칭 플랫폼 "StudyMate"

## 제품 개요
StudyMate는 대학생과 취준생이 관심사 기반으로 스터디 파트너를 빠르게 찾을 수 있는 AI 매칭 플랫폼이다.

## 핵심 문제
- 스터디 모집 공고를 올려도 적합한 사람을 찾는 데 평균 2주가 걸린다.
- 80%의 스터디가 첫 달 안에 해산한다 (목표/수준 미스매치 원인).
- 카카오오픈채팅/에브리타임은 매칭 알고리즘이 없어 수동 검색만 가능하다.

## 타겟 사용자
- 주요: 대학교 2~4학년, 취업 준비 중인 20대
- 부차: 직장인 사이드 프로젝트 팀 빌딩

## 핵심 기능 (v1.0 범위)
1. AI 스터디 매칭: 관심 분야, 목표, 일정, 수준을 입력하면 AI가 최적 파트너를 추천
2. 스터디 방 생성: 모집 공고 자동 생성 (AI 초안 작성 지원)
3. 진도 체크: 주간 체크인 리마인더 + 달성률 시각화
4. 채팅: 매칭된 그룹 내 실시간 채팅

## 제외 범위 (v1.0)
- 화상 회의 기능 (Zoom 연동으로 대체)
- 결제/프리미엄 플랜 (v2.0에서 검토)

## 성공 지표 (KPI)
- 매칭 후 4주 지속률 60% 이상
- 매칭까지 소요 시간 48시간 이하
- MAU 1,000명 (출시 3개월 내)

## 기술 스택
- Frontend: Next.js + TypeScript
- Backend: Supabase (DB + Auth + Realtime)
- AI: Gemini API (매칭 추천, 공고 초안 생성)
- 배포: Vercel

## 현재 상태
- 와이어프레임: 완성
- MVP 개발: 진행 중 (예상 완료: 2026년 7월)
- 베타 테스터 모집: 50명 대기 중
